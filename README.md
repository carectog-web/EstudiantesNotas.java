Objetivo

Aplicar el uso de colecciones de datos mediante un programa que permita almacenar, organizar y mostrar información.

 Problema
 
Se desarrolló un programa para registrar estudiantes y sus notas utilizando un diccionario (HashMap).
 Funcionamiento
 
El programa permite:

 Agregar estudiantes y sus notas
  Mostrar la lista de estudiantes
 Buscar un estudiante
  Eliminar un estudiante

Estructura utilizada

Se utilizó un **HashMap**, donde:

Clave: nombre del estudiante
Valor: nota

Ejecución

El programa funciona mediante un menú interactivo en consola.
